# Gra Przeglądarkowa – Zadanie Dodatkowe

## Cel zadania
Stwórz działającą grę przeglądarkową w **jednym pliku HTML**, którą można uruchomić po prostu klikając na plik w przeglądarce.

## Najważniejsze wymaganie
**Gra musi działać!**
Po otwarciu pliku w Chrome, Firefox lub Edge:
- można grać,
- zdobywać punkty,
- restartować grę.

## Jak możesz to zrobić?

### Opcja 1: Użyj AI
Najłatwiej skorzystać z narzędzi AI, np.:
- ChatGPT
- Copilot
- DeepSeek

Przykładowy prompt do AI:
> Napisz prostą grę węża (Snake) zjadającego przedmioty w jednym pliku HTML przy użyciu JavaScript, CSS i HTML.  
> Możesz też zaproponować inną prostą grę.

### Opcja 2: Znajdź gotowy kod
- Szukaj na GitHub: `"simple game html one file"`  
- Szukaj na CodePen: gry w jednym pliku HTML  

## Przykładowe gry
- Snake – wąż zjadający jabłka  
- Arkanoid – odbijanie piłki i niszczenie klocków  
- Flappy Bird – klikanie, aby ptak omijał przeszkody  
- Prosta wyścigówka – omijanie przeszkód  
- Memory – odkrywanie par obrazków  
- Tetris – układanie klocków  

## Wymagania techniczne
- Jeden plik, `twojeimie_gra.html`  
- Wszystko w jednym pliku (HTML, CSS, JavaScript)  
- Nie trzeba używać zewnętrznych bibliotek  
- Gra powinna działać po otwarciu w przeglądarce

## Krok po kroku
1. Użyj AI lub znajdź gotowy kod.  
2. Skopiuj cały kod.  
3. Wklej do nowego pliku `.txt` w Notatniku.  
4. Zmień rozszerzenie na `.html`.  
5. Kliknij dwukrotnie plik – gra powinna się otworzyć.  
6. Sprawdź działanie sterowania i zbierania punktów.

> Ważne: To zadanie sprawdza, czy potrafisz znaleźć lub wygenerować kod i go uruchomić. Nie trzeba pisać wszystkiego od zera – w pracy też korzysta się z gotowych rozwiązań.
