# Zadanie

## 🖊️ Opis gry

Stwórz grę składającą się z dwóch duszków: **duszek-balerina** 💃 oraz **duszek-jabłko** 🍎.

- **Sterowanie** → duszek-baleriną za pomocą klawiszy strzałek ←↑↓→.
- **Cel gry** → złapać jak najwięcej duszków-jabłek 🍎.
