## zadanie 3

Narysuj cztery kwadraty, ułożone jeden pod drugim, spełniające następujące warunki:

- Długość boku kwadratu: odpowiednio 10, 35, 60, 85.
- Odległość między kwadratami wynosi ```10 jednostek```.
